PROJECT
Hackathon Hawk

Hackathon Hawk is a mobile-responsive web application which allows users to search for upcoming hackathon events. While guest users can just search details of hackathons, member users can do more. They can bookmark hackathons, by which they can follow updates from the organizers. Also, they can partner with other member users to participate in a hackathon. Users can also post questions and notes to the organizers of the hackathon and other users.

    Technical Details
  • MEAN Stack, mobile responsive web application.
  • Used Bootstrap and jQuery for front-end beautification.
  • Used HackathonWatch API to search for the hackathons.
  • Used Passport.js for security. Enabled Facebook signup and login.

Assignment: Web App Maker

Web App Maker is a mobile responsive website builder web application. It allows users to create website with multiple pages and pages with multiple widgets. Widgets are draggable elements on a page like HTML elements, paragraph, heading, image, video, links, etc.

    Features
  • Users log in to use the application.
  • Users can create multiple websites
  • Each website has multiple pages.
  • Each page can have multiple widgets.
  • Examples of widgets: Image, video, HTML elements, heading, paragraph, etc.
  • Users can do a Flickr Image search.
